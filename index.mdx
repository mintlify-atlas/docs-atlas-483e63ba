GameAP is a free and open-source game server management panel that lets you deploy, monitor, and control game servers across multiple nodes — all from a single web interface. Whether you're running a single Minecraft server or a fleet of dedicated machines hosting dozens of game instances, GameAP gives you the tools to manage everything in one place.

## Get started in minutes

<Steps>
  <Step title="Deploy with Docker">
    Pull the GameAP image and start the panel with a single `docker-compose up -d` command. No pre-installed dependencies required.
  </Step>
  <Step title="Add a node">
    Install the GameAP Daemon on your game server host. The panel will guide you through the setup script.
  </Step>
  <Step title="Create a game server">
    Pick a game, configure ports and directories, and let GameAP install and launch the server automatically.
  </Step>
  <Step title="Manage from anywhere">
    Use the web UI or REST API to start, stop, update, and monitor your servers from any browser.
  </Step>
</Steps>

<Note>
  GameAP ships as a single binary with no external dependencies. You can run it on Linux, Windows, or macOS — and it supports PostgreSQL, MySQL, MariaDB, and SQLite as backing databases.
</Note>
